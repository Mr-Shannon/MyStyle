  @brief
    This is GD32 USB full speed device library.

    In order to use it, you should do as follows:

        - Download GD32 standard peripheral library and copy it to folder 
          "Libraries\GD32F10x_StdPeriph_Driver"

        - Maybe you need update the startup file "gd32f10x_startup.s" and 
          global header file "gd32f10x.h" if they are not latest 

        - You can directly run the demos in the folder "Projects"

        - You can also build your own project in the folder "Projects"
